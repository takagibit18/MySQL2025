表单功能说明
1. CSRF保护

表单中包含 {% csrf_token %}，这是Django的CSRF保护机制，防止跨站请求伪造攻击。

2. 表单验证

使用HTML5的 required 属性进行客户端验证，服务器端也会进行验证。

3. 表单提交

表单提交到 /api/login，使用POST方法，数据通过 request.POST 获取。

4. 表单处理

服务器端使用 @require_POST 装饰器确保只接受POST请求，使用 request.POST.get() 获取表单数据。

代码示例
# 视图函数
@require_POST
def login(request):
    student_id = request.POST.get("student_id")
    password = request.POST.get("password")
    # 处理登录逻辑...
    return HttpResponseRedirect("/ui/profile")

# 模板中的表单
<form method="post" action="/api/login">
    {% csrf_token %}
    <input name="student_id" required>
    <input name="password" type="password" required>
    <button type="submit">提交</button>
</form>
